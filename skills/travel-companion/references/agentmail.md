# AgentMail Integration

## Configuration

Set environment variable:
```
AGENTMAIL_API_KEY=your-key-from-agentmail.to
```

Get API key at: https://agentmail.to

## Send Email

```bash
# API endpoint
POST https://api.agentmail.to/v1/messages

# Headers
Authorization: Bearer {AGENTMAIL_API_KEY}
Content-Type: application/json

# Body
{
  "to": "user@example.com",
  "subject": "Trip Itinerary - Tokyo",
  "html": "<h1>Your Tokyo Trip</h1><p>Day 1: ...</p>"
}
```

## Create Inbox (Optional)

For receiving replies:
```json
{
  "action": "createInbox",
  "email": "user@example.com",
  "expiresInMinutes": 1440
}
```

## Email Template

```html
<h1>Your Trip to {Destination}</h1>

<h2>Weather</h2>
<p>{weather summary}</p>

<h2>Day 1</h2>
<ul>
  <li>{activity 1}</li>
  <li>{activity 2}</li>
</ul>

<h2>Day 2</h2>
<ul>
  <li>{activity 3}</li>
  <li>{activity 4}</li>
</ul>

<h2>Estimated Costs</h2>
<ul>
  <li>Flights: ${amount}</li>
  <li>Hotels: ${amount}</li>
  <li>Activities: ${amount}</li>
</ul>

<p>Generated by Travel Companion</p>
```
